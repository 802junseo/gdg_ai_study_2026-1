이번 2주차 Ai 정규 스터디를 통하여 데이터를 나누는 것과 표준화의 중요성을 알 수 있었다. 데이터를 나누는 것에도 특정 비율로 나누어 훈련 세트와 테스트 세트의 크기 차이가 모델의 성능의 어떤 영향을 끼치는지 확인 할 수 있었다. 그리고 데이터를 무작위로 섞어 테스트를 진행 할 수 있도록 하였고 평균, 분산, 표준편차등의 수학적 개념을 파이썬으로 구하며 과정을 이해할 수 있었고 마지막으로 데이터를 시각화 해 직접 눈으로 볼 수 있게 하여 단순히 숫자로 이루어진 수치가 아니라 직관적으로 통찰 할 수 있었다. 이를 통해 머신러닝 기초를 탄탄히 다질 수 있었다.
